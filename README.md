# Unificador de Links do Setor

Este repositório contém o código para uma página web simples e profissional que serve como um unificador de links para os sistemas do seu setor.

## Como colocar no ar (Passo a Passo)

Como você não quer instalar nada no seu computador, siga estes passos diretamente no site do GitHub:

1. **Crie um novo repositório:**
   - Acesse seu GitHub e clique no botão **"+"** (canto superior direito) e selecione **"New repository"**.
   - Dê um nome ao repositório (ex: `links-setor`).
   - Marque como **"Public"**.
   - Clique em **"Create repository"**.

2. **Crie o arquivo principal:**
   - Na página do novo repositório, clique no link **"creating a new file"**.
   - No campo do nome do arquivo, digite `index.html`.
   - No campo de conteúdo, cole o código do arquivo `index.html` que eu gerei para você.
   - Clique no botão verde **"Commit changes..."** no topo da página e depois em **"Commit changes"** novamente.

3. **Ative a página (GitHub Pages):**
   - No menu superior do seu repositório, clique em **"Settings"** (Configurações).
   - No menu lateral esquerdo, na seção "Code and automation", clique em **"Pages"**.
   - Em "Build and deployment" > "Branch", verifique se está selecionado **"main"** (ou `master`) e a pasta **"/(root)"**.
   - Clique em **"Save"**.

4. **Acesse sua página:**
   - Aguarde cerca de 1 a 2 minutos.
   - Recarregue a página de "Pages". Você verá uma mensagem dizendo: *"Your site is live at..."* seguido de um link.
   - Clique no link e sua página de sistemas estará no ar!

## Como personalizar os links

Para mudar os links e nomes dos sistemas:
1. Abra o arquivo `index.html` no GitHub.
2. Clique no ícone de lápis (**Edit this file**).
3. Procure pelas linhas que começam com `<a href="https://exemplo1.com"...`.
4. Altere o endereço (`href`), o título (`h3`) e a descrição (`p`) conforme sua necessidade.
5. Salve as alterações clicando em **"Commit changes"**.
